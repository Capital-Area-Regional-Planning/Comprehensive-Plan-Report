Q. How do I change the colors of the graphs or the formats of the tables?
A. You can style any graph or table as if it was created in Word.

Q. I don't want to show some of the data included in a graph
A. You can use Chart Filters to change what data shows in a graph. Alternatively, right-click on a chart and select 'Edit Data'. A pop-up window will show the underlying data that created the chart - edit this to change what data is included in the graph.

Q. I don't want to show some of the data included in a table
A. Simply delete any rows or columns you don't want to include.

Q. How do I change the graph style (i.e. make a bar chart into a pie chart)?
A. The easiest way to change the format of a chart is to right-click on a chart and select 'Edit Data'. Copy the data from the pop-up window and paste it into Excel. Create the desired graph in Excel, then copy/paste back into Word.

Q. Why is there a box around the graphs?
A. The border box is a quirk of the underlying code. To manually remove it, right-click on a graph and select 'Format Plot Area'. Select 'Chart Area' from the dropdown sidepanel and change 'Border' to 'No line'. To bulk remove the border from all graphs, follow these instructions.

Q. How accurate is this data?
A. The American Community Survey (ACS) surveys a sample of the U.S. population, so includes a margin of error for each estimate. The estimate, margin of error (MOE), coefficient of variation (CV), and interpreted reliability are reported for each variable in the Census Reporting section of the report. Data from the Decennial Survey is a complete population survey, so does not have a margin of error.
